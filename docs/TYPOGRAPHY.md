# Cinematic 3D Text Typography Presets

Software-agnostic settings you can apply in any 3D text application (After Effects, Cinema 4D, Blender, Three.js, etc.)

-----

## **FONT & TYPOGRAPHY SETTINGS**

### **Font Categories for Cinematic Text**

#### **1. Epic/Historical (Serif)**

- **Fonts**: Trajan Pro, Cinzel, Garamond, Didot
- **Best for**: Historical dramas, fantasy, epic narratives
- **Character**: Elegant, authoritative, timeless

#### **2. Modern/Action (Sans-Serif)**

- **Fonts**: Gotham, Futura, Bebas Neue, Montserrat, Poppins
- **Best for**: Action films, tech products, modern stories
- **Character**: Bold, clean, contemporary

#### **3. Horror/Thriller**

- **Fonts**: Distressed serifs with irregular baselines
- **Best for**: Suspense, psychological thrillers
- **Character**: Unsettling, threatening, deteriorating

#### **4. Sci-Fi/Futuristic**

- **Fonts**: Geometric sans-serifs, custom tech fonts
- **Best for**: Science fiction, technology
- **Character**: Sleek, angular, forward-looking

## **TYPOGRAPHY SPACING PRESETS**

### **Kerning (Space Between Letter Pairs)**

Kerning values are typically measured in units or as a percentage of em space:

- **Tight Cinematic**: -50 to -20 units
  - Creates dense, impactful titles
  - Use for: Action titles, powerful statements
- **Normal Cinematic**: -10 to +10 units
  - Balanced, professional spacing
  - Use for: Most title sequences
- **Loose Cinematic**: +20 to +100 units
  - Airy, elegant feel
  - Use for: Luxury brands, elegant credits

**Special Attention Letter Pairs:**

- "AV", "AW", "VA", "WA" - typically need -30 to -50
- "TO", "Yo", "Ty" - need -20 to -40
- Angular letters (A, V, W, Y) generally need more kerning

### **Tracking (Overall Letter Spacing)**

Tracking is uniform spacing applied across all characters:

- **Tight Tracking**: -30 to -10
  - Dense, bold look
  - Use for: Headlines with impact
- **Normal Tracking**: 0
  - Default spacing
  - Use for: Standard readability
- **Loose Tracking**: +50 to +200
  - Spacious, premium feel
  - Use for: All-caps titles, luxury brands
- **Extreme Wide**: +300 to +500
  - Dramatic, cinematic reveal effect
  - Use for: Title sequences, special effects

### **Leading / Line Height**

Leading is measured as percentage of font size or in points:

- **Tight Leading**: 100-110% of font size
  - Compact, stacked look
  - Use for: Multi-line titles with impact
- **Standard Leading**: 120-140% of font size
  - Industry standard
  - Use for: Most body text and titles
- **Loose Leading**: 150-200% of font size
  - Spacious, elegant
  - Use for: Artistic titles, poetry-style text

**Golden Rule**: Leading should be at least 20% greater than font size for readability

## **3D EXTRUSION & BEVEL SETTINGS**

### **Extrusion Depth Presets**

Measured in pixels or units relative to font size:

- **Subtle Depth**: 10-50px (or 5-10% of font height)
  - Light 3D effect
  - Use for: Elegant, refined titles
- **Medium Depth**: 100-300px (or 20-40% of font height)
  - Classic cinematic depth
  - Use for: Most 3D title work
- **Deep Extrusion**: 500-1000px (or 60-100% of font height)
  - Dramatic, sculptural look
  - Use for: Impact titles, logo reveals
- **Extreme Depth**: 1500px+ (or 150%+ of font height)
  - Architectural, dimensional
  - Use for: Abstract title sequences

### **Bevel Style & Depth Presets**

#### **Bevel Styles**

- **None**: Clean, sharp edges
- **Angular**: Hard, geometric bevels (5-15px depth)
  - Best for: Modern, tech, action
- **Concave**: Inward-curving bevels (10-20px depth)
  - Best for: Elegant, sophisticated
- **Convex**: Outward-curving bevels (10-25px depth)
  - Best for: Soft, organic, friendly
- **Curve/Rounded**: Smooth, rounded edges (15-30px depth)
  - Best for: Approachable, organic

#### **Bevel Depth Values**

- **Subtle**: 2-5px
  - Barely visible, refined
- **Standard**: 5-10px
  - Industry baseline
- **Pronounced**: 15-25px
  - Clear edge definition
- **Extreme**: 30-50px
  - Sculptural, artistic

#### **Hole Bevel Depth** (for letters like O, D, P)

- **Standard**: 50-100% of main bevel depth
  - Keeps inner holes clear and readable
- **Reduced**: 0-25%
  - When holes closing up, reduce this value

## **ADVANCED TYPOGRAPHY TIPS**

### **All-Caps Typography Rules**

- Always increase tracking: +100 minimum
- Reduce font size slightly (95% of mixed-case equivalent)
- Consider increasing leading by 10-20%

### **Font Weight Guidelines**

- **Thin/Light (100-300)**: Elegant, use larger sizes (150px+)
- **Regular (400-500)**: Versatile, standard use
- **Bold (600-700)**: Impactful, cinematic default
- **Black/Heavy (800-900)**: Maximum impact, use sparingly

### **Readability Checks**

1. **Squint Test**: Blur your eyes--text should still be recognizable
1. **Distance Test**: View from 10ft away--hierarchy should be clear
1. **Upside Down Test**: Flip text upside down to check spacing balance
1. **Different Screens**: Test on various devices/resolutions

### **Animation Considerations**

- Tighter tracking for fast reveals (then expand)
- Looser tracking for slow, elegant entrances
- Consider animating kerning for emphasis
- Extrusion depth can animate from 0 to final value

## **ASPECT RATIO ADJUSTMENTS**

Different aspect ratios may require typography tweaks:

**16:9 (Standard HD)**

- Use presets as-is

**21:9 / 2.39:1 (Cinematic)**

- Reduce font size by 10%
- Increase tracking by 20%
- Consider horizontal text layout

**9:16 (Vertical/Mobile)**

- Increase font size by 15%
- Reduce tracking
- Stack text vertically

**1:1 (Square/Social)**

- Center-focused composition
- Medium tracking
- Consider circular text arrangements

## **SOFTWARE-SPECIFIC NOTES**

### **After Effects**

- Extrusion Depth: Found in Geometry Options
- Use Cinema 4D Renderer for true 3D
- Kerning: Character panel (Alt + Arrow keys)

### **Cinema 4D**

- Use MoText object for full typography control
- Enable "Show 3D GUI" for visual kerning
- Extrude Object for depth control

### **Blender**

- Text objects → Convert to Mesh
- Extrusion in Object Properties
- Edit mode for individual letter kerning

### **Three.js / WebGL**

- Use TextGeometry
- Extrusion controlled via `extrudePath` settings
- CSS-based kerning for 2D overlay text

## **WORKFLOW TIPS**

1. **Start with font selection** - This drives all other decisions
1. **Set tracking first** - Establishes overall texture
1. **Kern problem pairs** - Manually adjust awkward combinations
1. **Add depth last** - 2D must look good before adding 3D
1. **Test in context** - Always view with background/scene
1. **Save presets** - Document what works for future projects

The key to cinematic 3D text is **restraint and intentionality**. Don’t use extreme settings just because you can--every adjustment should serve the story and emotional tone of your project.
